1. open the DATABASE folder-> inside which there is the "shoopingcart.sql" file.

2. restore the db using the following command
mysqldump -u [your userName] -p shoopingcart.sql > "[PATH]/shoopingcart.sql".sql" 
replace [PATH] with DATABASE folder path.

3. run XAMPP and start the "apache" and "Mysql" servers.

4.go to a web browser and type localhost
